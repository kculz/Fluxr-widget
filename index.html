üì° Event Listeners

Listen to widget events globally

// Listen for widget opened window.addEventListener('flx_open', function() { console.log('Widget opened'); }); // Listen for successful transaction window.addEventListener('flx_send_success', function(event) { console.log('Airtime sent!', event.detail); }); // Listen for errors window.addEventListener('flx_error', function(event) { console.error('Error occurred', event.detail); });

flx_open

Fired when the widget is opened

flx_close

Fired when the widget is closed

flx_step

Fired when user moves to a different step (includes step number in event.detail)

flx_send_success

Fired when airtime is successfully sent (includes reference and amount in event.detail)

flx_payment_start

Fired when payment process starts

flx_payment_success

Fired when payment is successful

flx_error

Fired when any error occurs (includes error code in event.detail)
